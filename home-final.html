Product

Our product, vcrypto, is a set of high-performance C++ libraries that can be seamlessly integrated into customers' existing trading platforms to enhance latency and improve profitability. These libraries utilise custom DPDK drivers and a specialized TLS/TCP network stack to significantly outperform existing network components used for market data and order execution.

Currently, vcrypto supports market data and is in the process of incorporating an order-entry component.

Perkunas targets proprietary trading firms and hedge funds engaged in algorithmic HFT on cryptocurrency exchanges that operate on Amazon Web Services (AWS), such as Binance, Coinbase, Bybit, Huobi, and Kucoin. These firms employ software developers to build automated trading strategies that capitalize on price movements by sending electronic orders to the exchange. The speed at which these algorithms process data and execute trades—known as "tick-to-trade latency"—is crucial, often occurring in the low single-digit microseconds. This rapid execution is far beyond human capabilities, necessitating sophisticated algorithms for profitable trading.
